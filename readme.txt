Link between WooCommerce and the Print Science Designer to allow for product customization and online design

== Description ==

Link between WooCommerce and the Print Science Designer to allow for product customization and online design. This plugin requires WooCommerce.

The Print Science Designer allows your customer to create customized products based on templates you create. After the customer completes the new design, the Designer creates a high-resolution PDF which you can download and print. 



== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the 'Plugins' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the Designer API information via the tab ‘Personalization API’ in the ‘Settings’ menu 

== Frequently Asked Questions ==

= Does the Designer plugin require WooCommerce  =

Yes. You must have WooCommerce installed.

= Can I use the Designer plugin with a shopping cart other than WooCommerce  =

No. The Designer plugin works only with WooCommerce. For other shopping carts, you must use a plugin built specifically for that shopping cart system.

= Is the Print Science Designer free of charge to use  =

No. The WooCommerce plugin is free-of-charge to install and test. However, the Print Science Designer requires a paid subscription to use.
